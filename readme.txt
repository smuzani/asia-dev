# Asia Dev Academy Samples

Class content and notes. You may need to run `npm install` on some Node projects.

 * **forms:** Basic form reader using Node.js
 * **hello-express:** Basic functions of Express
 * **html:** Various HTML exercises and examples
 * **mocha:** Basic test function with Mocha.js
 * **more-express:** Some more exercises with Express
 * **movies-express:** Intermediate Express exercises, with apiDoc
 * **node:** Exercises with Node.js, including slackbot
 * **passport-js:** Basic hashing with mongoose and Passport.js
 * **short-url:** URL shortener with Node.js
 * **test-driven:** Test driven development with Mocha.js and Chai.js
